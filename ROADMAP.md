- HTML5

- JavaScript (Vanilla)
- Tailwind CSS

## 단계별 개발 계획

### 1단계: 프로젝트 초기 설정
- [ ] 프로젝트 폴더 구조 생성
- [ ] Tailwind CSS CDN 또는 CLI 설정
- [ ] 기본 HTML 템플릿 작성 (`index.html`)
- [ ] 기본 CSS 파일 생성 (`style.css`)
- [ ] JavaScript 파일 생성 (`main.js`)

```
resume/
├── index.html
├── style.css
├── main.js
└── assets/
    └── images/
        └── profile.jpg
```

---

### 2단계: 레이아웃 구조 설계
- [ ] 반응형 레이아웃 설계 (모바일 / 태블릿 / 데스크탑)
- [ ] 네비게이션 바 (섹션 이동 링크)
- [ ] 메인 콘텐츠 영역 구분

**섹션 구성:**
1. Hero (소개)
2. About (자기소개)
3. Skills (기술 스택)
4. Projects (프로젝트)
5. Experience (경력)
6. Education (학력)
7. Contact (연락처)

---

### 3단계: 각 섹션 구현

#### Hero 섹션
- [ ] 이름, 직함 표시
- [ ] 짧은 소개 문구
- [ ] CTA 버튼 (연락하기, 이력서 다운로드)
- [ ] 프로필 이미지 또는 아바타

#### About 섹션
- [ ] 자기소개 텍스트
- [ ] 간단한 개인 정보 (위치, 이메일, 경력 연수 등)

#### Skills 섹션
- [ ] 기술 카테고리별 분류 (Frontend / Backend / Tools)
- [ ] 기술 아이콘 또는 뱃지 표시
- [ ] 숙련도 표시 (선택 사항)

**예시 기술 목록:**
- Frontend: HTML, CSS, JavaScript, React
- Backend: Node.js, Python
- Tools: Git, GitHub, VS Code, Figma

#### Projects 섹션
- [ ] 프로젝트 카드 레이아웃
- [ ] 프로젝트명, 설명, 사용 기술, GitHub/배포 링크

**예시 프로젝트 3개:**
1. 개인 블로그 - 정적 사이트 생성기 활용
2. Todo 앱 - React + Local Storage
3. 날씨 앱 - 외부 API 연동

#### Experience 섹션
- [ ] 타임라인 형태의 경력 목록
- [ ] 회사명, 직책, 기간, 주요 업무

**예시:**
- 2023.03 ~ 현재 | (주)OOO | 프론트엔드 개발자
- 2021.07 ~ 2023.02 | (주)XXX | 웹 개발자

#### Education 섹션
- [ ] 학교명, 전공, 졸업연도

**예시:**
- 2017 ~ 2021 | OO대학교 | 컴퓨터공학과 졸업

#### Contact 섹션
- [ ] 이메일, GitHub, LinkedIn 링크
- [ ] 간단한 문의 폼 (이름, 이메일, 메시지)
